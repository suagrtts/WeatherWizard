# 🌦️ Weather Wizard

Weather Wizard is a modern Android application that provides users with real-time weather updates, personalized settings, and a clean, user-friendly interface. Designed for convenience and simplicity, it helps users stay informed about weather conditions while offering customizable preferences.

---

## 📱 Features

* 🔍 **User Authentication**

  * Secure login and session management
  * Profile display (name & email)

* 🌤️ **Weather Information**

  * View current weather conditions *(expandable for future API integration)*

* ⚙️ **User Preferences**

  * Toggle temperature unit (Celsius / Fahrenheit)
  * Switch time format (12-hour / 24-hour)

* 🔐 **Account Management**

  * Change password with validation
  * Logout with confirmation dialog

* 📍 **Location (Coming Soon)**

  * Save and manage preferred locations

* 🧩 **Modular UI**

  * Built using Fragments for better navigation and scalability

---

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/weather-wizard.git
   ```

2. Open the project in **Android Studio**

3. Build and run on an emulator or Android device
